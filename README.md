# Arabic Educational Video Skill Extraction

An AI-based framework for extracting educational skills from Arabic mathematics videos and aligning them with the Palestinian curriculum.

## Pipeline

1. Video Collection
2. Speech Recognition (Whisper)
3. Transcript Correction
4. Arabic NLP Processing
5. Skill Extraction
6. Curriculum Alignment
7. Evaluation

## Dataset

- Grade 3 Mathematics
- 74 Educational Videos
- Palestinian Curriculum

## Results

Coverage: 70.48%
Curriculum Completeness Index (CCI): 80.2%
